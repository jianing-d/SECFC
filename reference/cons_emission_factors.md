# Consumer Emission Factors Dataset

This dataset contains emission factors for various consumer goods and
services based on country-specific data.

## Usage

``` r
data(cons_emission_factors)
```

## Format

A data frame with columns:

- Country:

  Country name

- FactorName:

  Type of consumer emission factor (e.g., Clothing, Dining, Healthcare,
  etc.)

- Value:

  Emission value for the given factor
