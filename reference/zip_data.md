# ZIP Code Data for U.S. States

This dataset provides the ZIP code ranges for different states in the
United States.

## Usage

``` r
data(zip_data)
```

## Format

A data frame with the following columns:

- ID:

  Unique identifier

- ST:

  State abbreviation

- State_Name:

  Full name of the state

- Zip_Max:

  Maximum ZIP code in the range

- Zip_Min:

  Minimum ZIP code in the range
