# Food Emission Factors Dataset

This dataset contains food-related emission factors based on
country-specific data.

## Usage

``` r
data(food_emission_factors)
```

## Format

A data frame with columns:

- Country:

  Country name

- FactorName:

  Type of food emission factor (e.g., Dairy, Meat, Vegan, Vegetarian)

- Unit:

  character

- Value:

  Emission value for the given factor
