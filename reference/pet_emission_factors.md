# Pet Emission Factors Dataset

This dataset contains emission factors for pet ownership based on
country-specific data.

## Usage

``` r
data(pet_emission_factors)
```

## Format

A data frame with columns:

- Country:

  Character, name of the country or region

- Dog:

  Numeric, carbon emissions from dogs (kg CO2 per year)

- Cat:

  Numeric, carbon emissions from cats (kg CO2 per year)
