# Housing Emission Factors Dataset

This dataset contains housing-related emission factors based on
country-specific data.

## Usage

``` r
data(housing_emission_factors)
```

## Format

A data frame with columns:

- Country:

  Country name

- FactorName:

  Type of housing emission factor (e.g., Electricity, Heating_Oil,
  Natural_Gas, Water)

- Value:

  Emission value for the given factor
